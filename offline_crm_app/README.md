# CRM אופליין (ללא אינטרנט)

תיקיה זו מכילה אפליקציית CRM מקומית בדפדפן:

- ללא תלות באינטרנט (אין CDN ואין קריאות לרשת).
- שמירה מקומית באמצעות `localStorage`.
- ייצוא רשומות לקובץ CSV שנפתח באקסל.

## איך להריץ

פשוט לפתוח את `index.html` בדפדפן.

## מה יש כאן

- `index.html` – ממשק הטופס והטבלה.
- `styles.css` – עיצוב מקומי.
- `app.js` – כל הלוגיקה ב-JavaScript מקומי.
